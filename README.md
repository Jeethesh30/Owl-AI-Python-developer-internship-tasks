Task 2 – AI Text Summarization Web Application

Task 2 is a Flask-based web application that allows users to input text and generate a summarized version using a simple interface.

The application provides an interactive UI where users can:

• Enter text manually
• Upload a .txt file
• Select the number of sentences for the summary
• Choose the language
• Download the summarized output

Application Workflow
1. Home Page

The Home page introduces the Owl AI summarization system and provides a button to start the summarization process.

It serves as the landing page of the application.

2. Input Page

The Input page allows users to provide text for summarization.

Features available on this page include:

• Text input field
• File upload option for .txt files
• Summary length selection
• Language selection
• Dark mode toggle
• Processing spinner during summarization
• Summary history display

After submitting the form, the application processes the input and generates a summarized output.

Users can also:

• Copy the generated summary
• Download the summary
• View recently generated summaries.

3. Display Page

The Display page shows the user's input and provides navigation back to the input page.
It ensures that the processed information is clearly presented to the user.
